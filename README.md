# Customer Churn Prediction
This project predicts whether a customer will churn or not using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
